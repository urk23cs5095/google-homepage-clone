# Google Homepage Clone

## Overview

This project is a front-end clone of the Google homepage developed using HTML and CSS. The objective was to recreate the visual layout and styling of the original Google search page while focusing on responsive design, spacing, typography, and user interface structure.

## Features

* Google-inspired homepage layout
* Search bar design
* Navigation links
* Responsive design
* Clean and organized code structure

## Technologies Used

* HTML5
* CSS3

## Learning Outcomes

* Improved understanding of HTML page structure
* Enhanced CSS styling and positioning skills
* Practiced responsive web design principles
* Learned how to recreate real-world user interfaces
